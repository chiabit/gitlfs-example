# gitfls-example

This repository is an example of a Git repository using Git LFS.

## Download repository

You can clone the repository with the following command:

```bash
git clone git@github.com:chiabit/gitlfs-example.git
```

We add the following JPG image:

```bash
git lfs track "*.jpg"
```
